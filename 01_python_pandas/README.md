## 当前理解

变量是用来保存值的名字。

列表可以保存多个值，索引从 0 开始。

字典用键值对保存结构化信息。

if 判断会从上到下执行，遇到第一个满足条件的分支就停止。

return 会把函数里的计算结果返回到函数调用的位置。
